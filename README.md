Pipes
=====

Augusts's entry into the 1 Game Per Month Challenge.

<img src="screenshot.png?raw=true" alt="Pipes Screenshot" width="512" height="448">

Connect the pipe from the startig point to the end point.


Controls
========

 * **D-Pad:** move cursor
 * **B:** Place piece
